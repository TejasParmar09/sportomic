# Spotomic Frontend

React frontend application for the Spotomic dashboard.

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

The application will run on http://localhost:3000

## Build

```bash
npm run build
```

## Environment Variables

Create a `.env` file in the frontend directory:

```
VITE_API_URL=http://localhost:5000/api
```

## Features

- Dashboard with General, Booking, and Coaching tabs
- Real-time metrics and KPIs
- Revenue charts with Chart.js
- Filter by venue, sport, month, and year
- Responsive design
